## Описание работы

Лабораторная работа посвящена освоению системы контроля версий Git. В ходе работы выполнялись различные операции с ветками, коммитами, слияниями и разрешением конфликтов.

## Выполненные задания

### Основные задания:

1. **Добавление строки "теормин"** в файл secrets.txt и создание коммита
2. **Добавление строки "не теормин"** в файл secrets.txt и создание коммита  
3. **Переход на ветку 'gguf'**, добавление строки "теормин" и создание коммита
4. **Сравнение коммитов** - сохранение разницы между последними двумя коммитами в файл diff.txt
5. **Откат к 3-му коммиту** на ветке main, изменение файла secrets.txt и создание нового коммита
6. **Rebase ветки 'gguf'** на main с разрешением конфликтов
7. **Слияние веток feature1 и feature2** в main с разрешением конфликтов
8. **Создание ветки 'release'** от main и объединение изменений

### Дополнительные задания:

- **Создание аккаунта на GitHub** и оформление репозитория
- **Сохранение полной истории коммитов** в файл history.txt
